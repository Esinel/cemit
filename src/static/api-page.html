REST API walkthrough

So here are some things you have to know:
  1. API root: api/railways
  2. Data for specific train (example: k08): /api/railways/k08
  3. Filtering by specific field: /api/railways/k08?year=2018&threshold-status=Over...

    Datafields that you can filter with:
    • year
    • threshold-status (Over || Under)
    • workorder-id
    • deviation-type (Vindskjevhet || Sporviddefeil)
You can chain filters with "&" as usual.
